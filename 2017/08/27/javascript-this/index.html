javascript this

在学习JS的过程中，被this搞得晕头转向，为了防止忘掉这些知识点，写篇blog记录下

在函数中this到底取什么值，是在函数被调用的时候确定的，在声明和定义的时候无法确定

  1. 构造函数
    • 用于new一个对象的构造函数，this代表new出来的对象
    • 如果直接调用构造函数，则this是window
    • 在构造函数的prototype中(即在整个原型链中，this代表的都是当前对象的值)
  2. 函数作为对象的属性
    • 函数作为一个对象的属性且被调用时，this指向该对象
    • 如果没有被调用,this是window
  3. 函数用了call或者apply调用
    • this的值取传入对象的值
  4. 全局或调用普通函数
    • 全局条件下this永远是window
    • 普通函数调用时其中的this也都是window
    • 当函数作为一个对象的内部属性，其内部又定义了一个”子函数”的时候，”子函数”中的this让指向window
